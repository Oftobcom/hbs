# React.js Tutorial 01 — Минимальный "Hello HBS" для начинающих

Это **первый** урок серии по React.js в контексте проекта **HBS (Human Body Simulation)**.  
Мы создадим минимальное React-приложение, которое выводит приветствие и базовую информацию о проекте — симуляторе физиологии человека, который в будущем будет интегрирован с электронной медицинской картой (ЭМК) и Python-бэкендом (Whole Body Simulation).

Цель — быстро запуститься и подготовить интерфейс для управления симуляцией, отображения графиков и взаимодействия с медицинскими данными.

---

## 1. Установка инструментов

### 1.1. Установите Node.js (если ещё не установили)

- Скачайте и установите LTS-версию с [официального сайта](https://nodejs.org/)
- Проверьте установку:
  ```bash
  node --version   # например, v18.17.0
  npm --version    # например, 9.6.7
  ```

### 1.2. Установите create-react-app (или используйте Vite)

Мы будем использовать `create-react-app` для быстрого старта:

```bash
npm install -g create-react-app
```

---

## 2. Создание проекта

```bash
# Перейдите в директорию для фронтенд-проектов
cd hbs/frontend

# Создаём React-приложение
create-react-app hbs-dashboard

# Переходим в папку проекта
cd hbs-dashboard
```

---

## 3. Запуск минимального приложения

```bash
npm start
```

После запуска откроется браузер с адресом `http://localhost:3000`. Вы увидите стандартную страницу "Welcome to React".

---

## 4. Первый собственный компонент

Очистим стандартный код и создадим компонент `HBSDashboard`.

### 4.1. Замените содержимое `src/App.js`:

```jsx
// src/App.js
import React from 'react';
import './App.css';

function App() {
  return (
    <div className="App">
      <header className="App-header">
        <h1>🫀 Human Body Simulation (HBS)</h1>
        <p>Версия: 0.1.0</p>
        <p>Модель: целостная физиология человека + ДМЖП</p>
        <hr />
        <h2>📋 Электронная медицинская карта (ЭМК)</h2>
        <p>Здесь будет отображаться информация о пациенте: диагнозы, лабораторные показатели, история болезни.</p>
        <h2>⚙️ Панель управления симуляцией</h2>
        <p>Запуск расчёта гемодинамики, настройка параметров (сопротивление ДМЖП, адаптация лёгких), отображение графиков давлений, потоков, концентраций.</p>
        <button onClick={() => alert('Симуляция ещё не подключена. Ждите следующих уроков!')}>
          ▶ Запустить симуляцию
        </button>
      </header>
    </div>
  );
}

export default App;
```

## 6. Подготовка к будущей интеграции

В следующих уроках (`reactjs_tutorial_02.md` и далее) мы:

- Создадим компоненты для **отображения данных пациента** (ФИО, возраст, диагнозы, лабораторные показатели — билирубин, аммиак, альбумин и т.д.).
- Добавим **форму управления симуляцией**:
  - Выбор типа пациента (здоровый / малый ДМЖП / большой ДМЖП)
  - Ручной ввод сопротивления дефекта `R_vsd`
  - Чекбокс «адаптация лёгких» (`flow_dependent_lungs`)
  - Длительность симуляции (t_span)
- Подключим **HTTP клиент** (Axios или Fetch) для отправки запросов к FastAPI бэкенду (`/simulate`).
- Реализуем **интерактивные графики** (с помощью `recharts` или `plotly.js`) для отображения:
  - Давлений (системное, лёгочное, венозное)
  - Кровотоков (аортальный, лёгочный, шунт)
  - Объёмов камер сердца
  - Концентраций веществ в крови
- Добавим **таблицу установившихся значений** (средние показатели за последние N секунд).
- Интегрируем **WebSocket** для получения промежуточных результатов длительных симуляций.
